# Heart Disease Classification Project

This project uses a heart disease dataset to demonstrate:

1. Training and visualizing a Decision Tree Classifier
2. Analyzing overfitting by controlling tree depth
3. Training and evaluating a Random Forest Classifier
4. Interpreting feature importances
5. Evaluating model performance using cross-validation

## Files

- `main.py`: Python script implementing all five tasks.
- `heart.csv`: Dataset (ensure it's in the same directory).
- `README.md`: Project overview.

## Usage

```bash
pip install -r requirements.txt  # Optional
python main.py
